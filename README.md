# apprecoleccion
Proyecto de Ingeniería de Software sobre una App de Reciclaje por Puntos

Este proyecto tiene como objetivo aplicar conceptos de programación y Python para crear una solución que contribuya a la preservación del medio ambiente.

# Funcionalidades clave
- Registro de puntos de recolección: Los usuarios podrán reportar y marcar en un mapa los puntos donde se encuentra basura acumulada.
- Notificaciones y recordatorios: La aplicación enviará notificaciones y recordatorios a los usuarios sobre las fechas y horarios de recolección de basura, así como actualizaciones del estado de sus reportes.
- Seguimiento y registro de actividades: Se llevará un registro de las actividades de recolección de basura, incluyendo información como ubicación, fecha, hora y tipo de residuos recolectados.
- Gamificación y recompensas: Se podrán implementar características de gamificación para incentivar y recompensar la participación de los usuarios en la recolección de basura, como logros, niveles y recompensas virtuales.

Este proyecto fue desarrollado como parte de un proyecto de una App de Recolección de Basura utilizando Python. Agradecemos a todos los instructores y colaboradores que hicieron posible este proyecto.
